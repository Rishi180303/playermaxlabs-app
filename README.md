# Player Max Labs App

This is the React Native (Expo) app for Player Max Labs.

Player Max Labs helps coaches and student-athletes manage training in one place:
- coaches assign workouts
- players log what they did
- players submit post-workout feedback
- coaches review progress and adjust plans

## Current Status

This repo currently has the opening screen UI implemented (Expo app starter + custom styling).

## Tech Stack

- React Native
- Expo
- TypeScript
- Outfit font

## Design Tokens (current)

- Background: `#080B18`
- Accent green: `#B4F469`

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start Expo:

```bash
npm run start
```

3. Run on platform:

- Press `i` for iOS simulator
- Press `a` for Android emulator
- Press `w` for web

Or run directly:

```bash
npm run ios
npm run android
npm run web
```
